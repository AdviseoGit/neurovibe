Senast uppdaterad: 22 april 2026

1. Inledning

Din integritet är fundamentet i Neurovibe. Vi samlar in minimalt med data och endast för att förbättra din upplevelse och vår forskning inom kognitivt stöd. Denna policy beskriver hur vi samlar in och använder din information.

2. Data vi samlar in

  • E-postadress: Om du anmäler dig till väntelistan samlar vi in din e-postadress. Denna används endast för att skicka inbjudningar och viktiga uppdateringar om plattformen.
  • Chattdata: Meddelanden du skriver till AI-Guiden bearbetas för att kunna generera ett svar. Vi sparar denna data anonymiserat för att förbättra tjänsten. Koppla aldrig känslig personlig information (som personnummer eller medicinska diagnoser kopplade till ditt namn) till chatten.
  • Användningsdata och Cookies: Vi använder nödvändiga cookies och kan komma att använda analysverktyg för att förstå hur webbplatsen används. Du kan styra detta via din webbläsare.

3. Tredjepartstjänster

Vi använder teknologier från OpenAI för att driva "Guiden". Ingen personligt identifierbar data (såsom din e-postadress) skickas dit tillsammans med dina meddelanden. Genom att använda chatten godkänner du att din inskrivna text bearbetas av OpenAI:s modeller.

4. Google AdSense

Vår webbplats är förberedd för Google AdSense. Tredjepartsleverantörer, inklusive Google, använder cookies för att visa annonser baserat på dina tidigare besök på vår eller andra webbplatser. Du kan välja bort personliga annonser genom att besöka Google Ads Settings.

5. Dina rättigheter

Du har rätt att när som helst begära att din e-postadress raderas från våra system eller begära ett utdrag av den data vi har om dig. Kontakta oss på simon@adviseo.se för alla ärenden gällande integritet.
